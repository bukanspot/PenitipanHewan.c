Judul : Penitipan hewan peliharaan
* Feature *
- Sistem Login
- Head Program ( tampilan awal )
- Pilihan Penitipan/Pengambilan hewan
- Asuransi hewan

[ Penitipan ]
• Input identitas penitip, hewan yg dititip & ket
• Hewan sakit tidak diterima
• Pilihan jenis hewan yang dititip:
  1.Anjing
  2.Kucing
  
[ Ras ]	 
  1.Small
  2.Medium
  3.Large
  4.Giant

• Pilihan paket penitipan:
[ Paket Penitipan ]
  1.Standar - Pemberian makan 3x sehari
  2.Spesial - (Standar) + Dimandikan 1x sehari
  3.Istimewa - (Spesial) + Dimandikan 2x sehari + vitamin setiap selesai makan 
• Pembuatan kode pengambilan hewan
• Pelanggan mendapat struk untuk pengambilan hewan

[ Pengambilan ]
• Input kode pengambilan hewan
• Melakukan pembayaran - cash only ( aplikasi kasir )
• Pelanggan mendapat struk pembayaran

[ Asuransi Hewan ]
• Apabila hewan sakit saat pengambilan, pembayaran hanya 50% dari seharusnya
• Apabila hewan mati di penitipan maka diganti rugi dengan  harga yg harusya dibayar pelanggan
